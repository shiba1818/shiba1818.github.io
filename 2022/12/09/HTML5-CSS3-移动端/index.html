HTML5+CSS3+移动端

发表于2022-12-09|更新于2022-12-15|web前端
|阅读量:

课程：Pink老师前端

学习路线

  1. HTML5基础，学习常见的没有兼容性的HTML标签。
  2. CSS3基础，学习常见没有兼容性的CSS样式，可利用标签实现传统PC端的网页布局。
  3. H5C3提高，学习HTML5新增的标签，CSS3样式。
  4. 综合案例：品优购电商网站

Web 标准的构成

  1. 结构(Structure)：结构用于对网页元素进行整理和分类，现阶段主要学的是 HTML。
  2. 表现(Presentation)：表现用于设置网页元素的版式、颜色、大小等外观样式，主要值 CSS。
  3. 行为(Behavior)：行为是指网页模型的定义及交互的编写，现阶段主要学的是 Javascript。

DOCTYPE和lang以及字符集的作用

  • <!DOCTYPE html> 文档声明标签，告诉浏览器这个页面采取HTML5版本显示页面
  • <html lang="en"> en表示英文显示，zh-CN表示中文显示
  • <meta charset="UTF-8">，表示字符集使用utf-8

标题标签<h1> - <h6>

段落标签和换行标签

  • 段落标签

段落标签可以将文字分段显示，它可以将网页分成若干个段落。

paragraph单词的缩写，意为段落。

特点：文本在一个段落中会根据浏览器大小自动换行。

  • 换行标签

标签语义：强制换行

特点：是一个单标签

文本格式化标签

标签语义：突出重要性，比普通文字更重要。

div和span标签

div和span是没有语义的，它们就是一个盒子，用来装内容的。

div是division的缩写，表示分割、分区。

span意为跨度、跨距。

  • div标签用来布局，一个div单独占一行，大盒子

  • span用来布局，一行上可以有多个<span>，小盒子

图像标签

img常用属性

属性 属性值 说明
src 图片路径 必须属性，本地相对路径或者绝对路径，或者完整的网络资源地址
alt 文本 替换文本，图片不能显示时显示的文字
title 文本 提示文本，鼠标放到图像上，现实的文字
width 像素 设置图像的宽度
height 像素 设置图像的高度
border 像素 设置图像的边框粗细

图像标签注意点：

  1. 图像标签可以有多个属性，必须写在标签名后面。
  2. 属性之间不分先后顺序，属性与属性之间以空格分开。
  3. 属性采用键值对的方式，及key="value"，属性="属性值"。

链接标签

在HTML标签中，<a>标签用于定义超链接，作用是从一个页面链接到另一个页面。

  1. 链接的语法格式及分类

属性 作用
href 用于指定链接目标的url地址，（必须属性）当为标签应用href属性时，他就有了超链接的功能。
target 用于指定链接页面的打开方式，其中_self为默认值，当前窗户口打开，_blank为在新窗口中打开方式。

表格标签

  1. 主要作用：表格主要用于显示、展示数据，因为它可以让数据显示的非常规整，可读性非常好。特别是后台显示数据的时候，能够熟练运用表格很重要。

  2. 基本语法：

    • table 用于定义表格的标签。
    • tr 用于定义表格中的行
    • td 表示表格数据(table data)，即数据单元格的内容
    • th表示HTML表格的表头部分(table head)
